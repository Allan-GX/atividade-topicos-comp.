- git init -> inicia um repositório.
- git clone -> clona um repositório.
- git add -> aisa todas as alterações feitas antes de dar um commit.
- git commit -m -> salva alterações dos arquivos e adiciona uma mensagem.
- git push -> envia alterações para o remoto.
- git pull -> atualiza o repositório local.
- git branch nome-da-branch -> cria uma nova branch.
- git branch -d nome-do-branch -> deleta uma branch.
- git switch -> troca de branch.
- git status -> mostra o estado do repositório.
- git merge nome_da_branch -> mescla branches.
- git stash -> salva mudanças não commitadas.
- git revert commit_id -> desfaz um commit.
- git log -> histórico de commits.
- git rebase nome_da_branch -> reaplica commits.
- git remote add origin link -> conecta ao remoto.
