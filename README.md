========================================================================================================
git init -> inicia um repositório.
git clone -> clona um repositório.
git add -> aisa todas as alterações feitas antes de dar um commit.
git commit -m -> salva alterações dos arquivos e adiciona uma mensagem para esse ponto de salvamento.
git push -> envia alterações feitas do repositorio local para o remoto. 
git pull -> atualiza o repositorio local com alterações do repositorio remoto.
git branch nome-da-branch -> cria uma nova branch.
git branch -d nome-do-branch -> deleta uma branch de forma segura.
git switch -> usado para trocar de branch.
git status -> Fornece informações sobre o estado atual do seu repositório.
git merge nome_da_branch -> usado para mesclar alterações de diferentes branches.
git stash -> salva mudanças que ainda não foram comitadas.
git revert commit_id -> cria um novo commit que desfaz as mudanças introduzidas por um commit anterior.
git log -> exibe o histórico de commits de um repositório.
git rebase nome_da_branch -> Reaplica a commit de uma branch sobre outra.
git remote add origin link-> conecta seu repositorio local com o remoto.
